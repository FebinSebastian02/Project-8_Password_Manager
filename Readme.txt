PASSWORD_MANAGER

Steps done:-
1) Functions were defined and called.
2) A while loop was used to repeat the program continuously until the user needs to exit.
3) with keyword was used to automatically close the file once operations were performed.
4) open() used to open a file and close() used to manually close a file.
5) if else conditions were written to execute statements based on user's choice.
6) cryptography package was installed for using the Fernet module for encryption.
7) A key is generated once with the help of Fernet module to encrypt the password.
8) 2 files - a) Password.txt and b) Key.key are created
9) Password when added in Password.txt file is encrypted using encrypt()
10) Password when viewed from Password.txt file is decrypted using decrypt()

Status of the project:- Completed
